Watermark 3D

Онлайн сервис для защиты 3D файлов

Сервис добавляет в 3D файлы специальные водяные знаки, которые нельзя убрать, без разрушения самого 3D файла.

  • HTML&CSS верстка макетов под десктоп и мобилки (адаптив)

Основа проекта

Проект начинался с бэкенд решения без интерфейса. Была только дефолтная форма загрузки модели и проверки водяного знака.

Исходя из базового функционала начался дизайн всего проекта. Основой которого стала сама форма для создания и проверки водяного знака. Сайт сделан одностраничным, с простой навигацией по странице.

Были созданы дополнительные страницы для FAQ и API.
